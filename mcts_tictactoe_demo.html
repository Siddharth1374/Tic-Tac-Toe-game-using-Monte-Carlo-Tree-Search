MCTS Phases

Phase 1
Selection
Traverse tree from root using UCB1 to pick the most promising node to explore.
Phase 2
Expansion
Add a new child node representing an unexplored move from the selected node.
Phase 3
Simulation
Play out a random game ("rollout") from the new node to a terminal state.
Phase 4
Backpropagation
Update win/visit counts from the terminal node back up to the root.

UCB1 Formula
w/n + C·√(ln N / n)
w = wins from this node
n = visits to this node
N = parent visits
C = √2 (exploration constant)

Balances exploiting known good moves vs exploring unknown ones.
